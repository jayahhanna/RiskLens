# RiskLens

RiskLens is an interactive portfolio risk analytics dashboard that helps users analyze portfolio exposure and visualize risk metrics using real market data.

The application allows users to enter multiple holdings and instantly evaluate key portfolio risk indicators including volatility, Value at Risk (VaR), concentration risk, and an overall portfolio risk score.

RiskLens integrates live financial data through an external market data API and performs risk calculations directly in the browser using JavaScript. The dashboard also includes visual tools such as allocation charts and a dynamic risk gauge to help interpret portfolio risk more clearly.

## Features

- Multi-asset portfolio input
- Live market data integration
- Portfolio value calculation
- Annualized volatility analysis
- 1-Day Value at Risk (95%)
- Concentration risk evaluation
- Dynamic risk score visualization
- Portfolio allocation chart
- Persistent holdings using localStorage
- API response caching to reduce redundant requests

## Technologies Used

- HTML
- CSS
- JavaScript
- Chart.js
- External market data API
- Browser localStorage

## Project Purpose

RiskLens was developed to explore portfolio risk analytics concepts and demonstrate how financial data, risk metrics, and visualization can be combined into an interactive analytical dashboard.

The project focuses on translating complex portfolio risk concepts into a clear and interpretable interface.
